# Build your own useless machine with Arduino

This is another useless machine project. This one is specially designed to for people who would like to learn coding. With this useless machine you've directly got a working machine with input output and analog out.

Here You'll find the information for building a arduino based useless machine
In the drawings foder you'll find the lasercutter drawings and stl file and 3d drawing for the mounting clips.

For the arduino project I've created a simple timer:
https://github.com/codecollective/arduino_simple-timer
